HTML[HyperText Markup Language] - язык разметки(потому что нет памяти и переменных как на python)
Используется чтобы расставить элементы(кнопки, фотки и т.д.) на сайте 



<название_тега></название тега> - так строятся теги(некоторые пишутся без </>)



<!DOCTYPE html> - указываем, что файл будет работать с html

<html></html> - главный тег, где указывается весь код

<head></head> - указываются параметры сайта(иконка, title, meta-данные)

<meta> - метаданные сайта, которые указывают его значения. например:
    *автор
    *описание
    *ключевые слова для поиска
    *формат текста

<body></body> - указывается содержимое самого сайта

<title></title> - указывает заголовок сайта, который пишется во вкладке

<h1></h1>(header. может быть и h2. и так до h6 [h1, h2, h3, h4, h5, h6]) - 
    создаёт заголовок на сайте, определяя части и деления текста. 
    по увеличению нумерации, текст уменьшается. например:
        h1 > h2 > h3 > h4 > h5 > h6

<p></p>(paragraph) - текст на сайте

<b></b>(bold) - жирный текст

<i></i>(italic) - курсивный текст

<u></u>(underlined) - подчёркнутый текст

<ul></ul>(unordered list) - маркированный список без нумерации(очереди)

<ol></ol>(ordered list) - нумерованный список с нумерацией(очередью)

<li></li>(list item) - компонент в списке

<header></header>(верхушка) - отвечает за заголовок и порой хранит навигацию

<nav></nav>(навигация) - блок для построения навигации

<main></main>(основа) - основной контект сайта

<footer></footer>(низ) - отвечает за нижнюю часть, рекламу и копирайт

<section></section>(область) - создает какую-то область на нашем сайте

<script></script>(скрипт) - назначает джаваскрипт(javascript) скрипт для работы с анимациями и интерактивом на странице

<video></video>(видео) - загружает видео-файлы на сайта

<source>(источник) - ссылка или расположение

<iframe></iframe>(картинка) - по ссылке загружает видео на сайт при указании нужных аргументов
