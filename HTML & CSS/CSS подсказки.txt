CSS[Cascade Style Sheet] - язык каскадных стилей
Используется чтобы указать стили элементам(которые указаны в HTML) на сайте 

название_тега {
    параметр_стиля: значение_стиля
}

основной_тег > второстепенный_тег - все второстепенные теги которые в основном

Иерархия в стилизации(т.е. важность или же уровни)
1) id [идентификатор]
2) class [классы]
3) <тег> [теги]
4) * [всё]
